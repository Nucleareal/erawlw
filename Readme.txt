;はじめに
;--------------------------------------------------
このバリアント(erawlwN)はerakanonをベースにwlwのキャラを追加し、手を加えたものです。
Emueraで動かされる前提のバリアントなので、eramakerでは動かすことができません。
このゲームは成年向けの二次創作です。18歳未満の方は直ちにこのファイルがあるフォルダごと削除して下さい。
なお、このゲームの元となった
・eramaker及びerakanonの制作元サークルである「漠々ト、獏」
・Wonderland Warsの制作元企業である「SEGA」
についてサポートを求めないで下さい。
本ゲームはフィクションであり、実在の人物や団体、地名等と一切関係ありません。

;--------------------------------------------------
; erawlwNについて
;--------------------------------------------------
このゲームは監禁・調教というハードな内容を含みますので、人の目に付くところでの話題は控えていただきますようお願いします。
